Take advantage of a vulnerability in the <color=#0FABFFFF>ssh</color> service to inject a new password to a registered user.
<b>Remote use.</b>

<b>Permissions obtained: </b>Non root user
<b>Target: </b> libssh.so >= v1.0.0

<b>Required:</b>
- Dependencies on library net.so >= 1.0.0
- Root user logged in the computer.=Использует уязвимость в сервисе <color=#0FABFFFF>ssh</color> для внедрения нового пароля зарегистрированному пользователю.
<b>Удаленное использование.</b>

<b>Полученные разрешения: </b>Пользователь без root-прав
<b>Цель: </b> libssh.so >= v1.0.0

<b>Требуется:</b>
- Зависимость от библиотеки net.so >= 1.0.0
- Root-пользователь, авторизованный на компьютере.
