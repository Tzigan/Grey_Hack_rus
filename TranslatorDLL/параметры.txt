В ОС Windows логи Unity обычно находятся в:
•	%USERPROFILE%\AppData\LocalLow\[Developer Name]\Grey Hack\Player.log
Если есть возможность запустить игру с параметром -logfile output.log, то все отладочные сообщения будут записаны в этот файл.


# Сначала соберите библиотеку
dotnet build GreyHackTranslator.csproj -c Release

# Затем соберите инжектор
dotnet build GreyHackInjector.csproj -c Release
